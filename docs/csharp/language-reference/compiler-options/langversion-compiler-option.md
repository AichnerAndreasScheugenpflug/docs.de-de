 <span data-ttu-id="62334-135">Da jede Version des C#-Compilers Erweiterungen der Sprachspezifikation enthält, bietet **-langversion** Ihnen nicht die gleiche Funktionalität wie die einer früheren Compilerversion.</span><span class="sxs-lookup"><span data-stu-id="62334-135">Because each version of the C# compiler contains extensions to the language specification, **-langversion** does not give you the equivalent functionality of an earlier version of the compiler.</span></span>

 <span data-ttu-id="62334-140">Weitere Möglichkeiten zum Angeben der C#-Sprachversion finden Sie unter [Select the C# language version (Auswählen der C#-Sprachversion)](../configure-language-version.md).</span><span class="sxs-lookup"><span data-stu-id="62334-140">For other ways to specify the C# language version, see the [Select the C# language version](../configure-language-version.md) topic.</span></span>
