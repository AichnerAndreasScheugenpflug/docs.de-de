# <a name="ltpropertygt-parameters-cannot-be-declared-byref"></a>&lt;Eigenschaft&gt; Parameter können nicht 'ByRef' deklariert werden
Parameter für Eigenschaften müssen mit dem `ByVal` -Modifizierer deklariert werden.  
  
 **Fehler-ID:** BC30651  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie Eigenschaftenparameter als `ByVal`.
