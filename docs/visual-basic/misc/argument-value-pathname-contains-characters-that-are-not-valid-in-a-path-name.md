# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>Der Argumentwert '&lt;Pfadnamen&gt;"enthält Zeichen, die in einem Pfadnamen nicht gültig sind
Ein Pfadname enthält ungültige Zeichen.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie das Zeichen aus dem Pfadnamen.
