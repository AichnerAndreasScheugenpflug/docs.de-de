# <a name="type-lttypenamegt-cannot-be-used-as-an-attribute-because-it-is-not-declared-public"></a>Typ "&lt;Typename&gt;' kann nicht als Attribut verwendet werden, da er nicht auf"Public"deklariert ist
Die Attribute müssen öffentlich sein.  
  
 **Fehler-ID:** BC31516  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Fügen Sie der Attributklasse den `Public` -Modifizierer hinzu.
