# <a name="ltmembername1gt-cannot-override-ltmembername2gt-because-they-differ-by-the-types-of-optional-parameters"></a>"&lt;membername1&gt;"darf nicht außer Kraft setzen"&lt;membername2&gt;" da sich die Typen der optionalen Parameter unterscheiden
Überladene Member müssen für mindestens einen erforderlichen Parameter unterschiedliche Datentypen aufweisen.  
  
 **Fehler-ID:** BC30697  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Definieren Sie mindestens einen erforderlichen Parameter als anderen Datentyp.
