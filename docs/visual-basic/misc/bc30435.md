# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Member in einer Struktur können nicht deklariert werden kann '\<Spezifizierer >'
Sie haben einen Spezifizierer verwendet, der für einen Member innerhalb einer `Structure` -Anweisung ungültig ist.  
  
 **Fehler-ID:** BC30435  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Spezifizierer.
