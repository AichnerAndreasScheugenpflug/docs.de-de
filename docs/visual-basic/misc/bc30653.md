# <a name="reference-required-to-module-modulename-containing-the-type-membername"></a>Verweis auf das Modul erforderlich "\<Modulname >' mit dem Typ '\<Membername >'
Verweis auf das Modul erforderlich "\<Modulname >' mit dem Typ '\<Membername >'. Fügen Sie dem Projekt einen Verweis hinzu.  
  
 Das Projekt enthält einen Typ, der von einem anderen Typ aus einer Assembly abhängt, die nicht in den Projektverweisen enthalten ist.  
  
 **Fehler-ID:** BC30653  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie einen Projektverweis für die abhängigen Typen im Projekt hinzu.
