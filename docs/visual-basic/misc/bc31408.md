# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>"&lt;spezifizierer1&gt;'und'&lt;spezifizierer2&gt;' können nicht kombiniert werden
Zwei inkompatible Spezifizierer, wie z. B. `Shadows` und `Overrides` , werden zusammen verwendet.  
  
 **Fehler-ID:** BC31408  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie einen der Spezifizierer.
