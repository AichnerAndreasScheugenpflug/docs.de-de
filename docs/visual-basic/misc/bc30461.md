# <a name="classes-cannot-be-declared-ltspecifiergt"></a>Klassen können nicht deklariert werden kann '&lt;Spezifizierer&gt;"
Sie haben versucht, eine `Class` oder ein `Module` mit einem ungültigen Spezifizierer zu deklarieren.  
  
 **Fehler-ID:** BC30461  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Entfernen Sie den Spezifizierer.
