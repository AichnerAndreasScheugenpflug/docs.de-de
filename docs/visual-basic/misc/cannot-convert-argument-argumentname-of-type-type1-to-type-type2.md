# <a name="cannot-convert-argument-ltargumentnamegt-of-type-lttype1gt-to-type-lttype2gt"></a>Argument kann nicht konvertiert werden kann '&lt;Argumentname&gt;"vom Typ"&lt;type1&gt;' in Typ'&lt;Typ2&gt;"
Eine Konvertierungsoperation versucht, ein Argument in einen inkompatiblen Typ zu konvertieren.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass der Typ, in den Sie das Argument konvertieren möchten, gültig ist.
