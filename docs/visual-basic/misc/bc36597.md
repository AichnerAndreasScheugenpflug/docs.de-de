# <a name="goto-to-label-ltlabelnamegt-cannot-be-used-within-lambda-expressions"></a>GoTo zu Bezeichnung "&lt;Bezeichnungsname&gt;' kann nicht in Lambdaausdrücken verwendet werden
Der Kern eines Lambda-Ausdrucks muss ein Ausdruck sein, der einen einzelnen Wert berechnet und zurückgibt. Es sind keine Anweisungen zulässig, wozu auch `GoTo` -Anweisungen gehören.  
  
 **Fehler-ID:** BC36597
