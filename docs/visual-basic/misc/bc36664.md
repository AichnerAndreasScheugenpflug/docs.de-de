# <a name="nesting-of-parentheses-in-lambda-expressions-parameter-list-exceeds-maximum-allowable-depth"></a>Die Anzahl der geschachtelten runden Klammern in der Parameterliste des Lambdaausdrucks übersteigt die maximal zulässige Tiefe.
Für geschachtelte Klammern in der Parameterliste eines lambda-Ausdrucks darf eine Tiefe von 128 nicht überschritten werden.  
  
 **Fehler-ID:** BC36664
