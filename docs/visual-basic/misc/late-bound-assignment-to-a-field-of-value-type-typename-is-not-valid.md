# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>Spät gebundene Zuweisung an ein Feld des Werttyps '&lt;Typename&gt;'ist ungültig, wenn'&lt;Namen&gt;"ist das Ergebnis eines spät gebundenen Ausdrucks
Sie haben versucht, eine spät gebundene Zuweisung zu verwenden, die nicht gültig ist.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Ändern Sie die Zuweisung in eine früh gebundene Zuweisung.
