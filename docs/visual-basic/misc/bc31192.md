# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Fehler beim Einbetten des Win32-Manifests: Manifestdatei &lt;Datei&gt; wurde nicht gefunden
Die für die Compileroption `/win32manifest` angegebene Datei wurde nicht gefunden.  
  
 **Fehler-ID:** BC31192  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Geben Sie einen gültigen Dateinamen und Pfad für die `/win32manifest` -Compileroption an.
