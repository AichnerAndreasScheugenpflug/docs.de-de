# <a name="line-number-cannot-be-parsed-using-the-current-delimiters"></a>Zeile \<Nummer > kann nicht mit den aktuellen Trennzeichen analysiert werden
Die angegebene Zeile kann nicht analysiert werden, da sie andere als die angegebenen Trennzeichen verwendet.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Passen Sie `Delimiters` so an, dass die Zeile fehlerfrei analysiert werden kann, oder f√ºgen Sie Ausnahmebehandlungscode ein, um die Zeile zu verarbeiten.
