# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>Verursacht einen Konflikt mit "\<name1 > ', der implizit für'\<name2 >' in \<Typ > '\<name3 >'
Der Name eines Typmembers steht in Konflikt mit einem anderen implizit erstellten Member.  
  
 **Fehler-ID:** BC31058  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Benennen Sie zur Behebung des Namenskonflikts den explizit deklarierten Member um.
