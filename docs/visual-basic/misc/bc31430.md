# <a name="expression-of-type-lttypename1gt-can-never-be-of-type-lttypename2gt"></a><span data-ttu-id="9ce41-102">Ein Ausdruck vom Typ "&lt;Typname1&gt;"kann niemals gleich vom Typ"&lt;Typname2&gt;"</span><span class="sxs-lookup"><span data-stu-id="9ce41-102">Expression of type '&lt;typename1&gt;' can never be of type '&lt;typename2&gt;'</span></span>
<span data-ttu-id="9ce41-103">Ein `TypeOf`...`Is` -Ausdruck überprüft eine Objektverweisvariable auf einen Datentyp, den er nicht enthalten kann.</span><span class="sxs-lookup"><span data-stu-id="9ce41-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="9ce41-104">In manchen Fällen kann der Compiler bestimmen, dass ein `TypeOf`...`Is` -Test zwangsläufig fehlschlagen muss, z. B. wenn zwischen zwei Klassen keine Vererbungsbeziehung besteht.</span><span class="sxs-lookup"><span data-stu-id="9ce41-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="9ce41-105">Dieser Fehler kann durch folgenden Code generiert werden.</span><span class="sxs-lookup"><span data-stu-id="9ce41-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="9ce41-106">Da <xref:System.Windows.Forms.Form> und <xref:System.Array> völlig voneinander unabhängige Typen sind, kann der Compiler bestimmen, dass der Ausdruck `TypeOf`...`Is` für jeden Wert von `False` den Wert `refVar`zurückgibt.</span><span class="sxs-lookup"><span data-stu-id="9ce41-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="9ce41-107">**Fehler-ID:** BC31430</span><span class="sxs-lookup"><span data-stu-id="9ce41-107">**Error ID:** BC31430</span></span>
