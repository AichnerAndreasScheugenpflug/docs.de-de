# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>Warnungsnummer "&lt;Anzahl&gt;"für die Option"&lt;Optionsname&gt;' ist entweder nicht konfigurierbar oder ungültig
Die ID einer Warnung ist ungültig. IDs müssen ganzzahlige Werte größer als 0 sein.  
  
 **Fehler-ID:** BC2026  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Überprüfen Sie die angegebene Warnungs-ID, und stellen Sie sicher, dass diese gültig ist.
