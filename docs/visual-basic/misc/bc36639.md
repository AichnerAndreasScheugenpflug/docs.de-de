```

## <a name="to-correct-this-error"></a><span data-ttu-id="079d2-106">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="079d2-106">To correct this error</span></span>

        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
