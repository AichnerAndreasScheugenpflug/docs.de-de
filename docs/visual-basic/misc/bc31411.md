# <a name="lttypenamegt-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>"&lt;Typename&gt;' müssen werden als" MustInherit"deklariert, da sie die Methoden, die als"MustOverride"enthält
Typen, die `MustOverride` -Member enthalten, müssen als `MustInherit`deklariert werden.  
  
 **Fehler-ID:** BC31411  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
-   Fügen Sie dem Typ den `MustInherit` -Modifizierer hinzu.
