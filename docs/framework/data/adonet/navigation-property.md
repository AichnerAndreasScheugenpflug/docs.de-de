# <a name="navigation-property"></a><span data-ttu-id="19b57-102">Navigationseigenschaft</span><span class="sxs-lookup"><span data-stu-id="19b57-102">navigation property</span></span>
<span data-ttu-id="19b57-103">Ein *Navigationseigenschaft* ist eine optionale Eigenschaft auf eine [Entitätstyp](../../../../docs/framework/data/adonet/entity-type.md) ermöglicht, die die Navigation von einer [End](../../../../docs/framework/data/adonet/association-end.md) von einer [Zuordnung](../../../../docs/framework/data/adonet/association-type.md) auf das andere Ende.</span><span class="sxs-lookup"><span data-stu-id="19b57-103">A *navigation property* is an optional property on an [entity type](../../../../docs/framework/data/adonet/entity-type.md) that allows for navigation from one [end](../../../../docs/framework/data/adonet/association-end.md) of an [association](../../../../docs/framework/data/adonet/association-type.md) to the other end.</span></span> <span data-ttu-id="19b57-104">Im Gegensatz zu anderen [Eigenschaften](../../../../docs/framework/data/adonet/property.md), Navigationseigenschaften werden keine Daten enthalten.</span><span class="sxs-lookup"><span data-stu-id="19b57-104">Unlike other [properties](../../../../docs/framework/data/adonet/property.md), navigation properties do not carry data.</span></span>  
  
 <span data-ttu-id="19b57-105">Die Definition einer Navigationseigenschaft enthält Folgendes:</span><span class="sxs-lookup"><span data-stu-id="19b57-105">A navigation property definition includes the following:</span></span>  
  
-   <span data-ttu-id="19b57-106">Einen Namen.</span><span class="sxs-lookup"><span data-stu-id="19b57-106">A name.</span></span> <span data-ttu-id="19b57-107">(erforderlich)</span><span class="sxs-lookup"><span data-stu-id="19b57-107">(Required)</span></span>  
  
-   <span data-ttu-id="19b57-108">Die Zuordnung, in der sie navigiert.</span><span class="sxs-lookup"><span data-stu-id="19b57-108">The association that it navigates.</span></span> <span data-ttu-id="19b57-109">(erforderlich)</span><span class="sxs-lookup"><span data-stu-id="19b57-109">(Required)</span></span>  
  
-   <span data-ttu-id="19b57-110">Die Enden der Zuordnung, in der sie navigiert.</span><span class="sxs-lookup"><span data-stu-id="19b57-110">The ends of the association that it navigates.</span></span> <span data-ttu-id="19b57-111">(erforderlich)</span><span class="sxs-lookup"><span data-stu-id="19b57-111">(Required)</span></span>  
  
 <span data-ttu-id="19b57-112">Beachten Sie, dass Navigationseigenschaften für beide Entitätstypen an den Enden einer Zuordnung optional sind.</span><span class="sxs-lookup"><span data-stu-id="19b57-112">Note that navigation properties are optional on both entity types at the ends of an association.</span></span> <span data-ttu-id="19b57-113">Wenn Sie für einen Entitätstyp am Ende einer Zuordnung eine Navigationseigenschaft definieren, muss keine Navigationseigenschaft für den Entitätstyp am anderen Ende der Zuordnung definiert werden.</span><span class="sxs-lookup"><span data-stu-id="19b57-113">If you define a navigation property on one entity type at the end of an association, you do not have to define a navigation property on the entity type at the other end of the association.</span></span>  
  
 <span data-ttu-id="19b57-114">Der Datentyp einer Navigationseigenschaft richtet sich nach der [Multiplizität](../../../../docs/framework/data/adonet/association-end-multiplicity.md) von dessen Remote [Zuordnungsende](../../../../docs/framework/data/adonet/association-end.md).</span><span class="sxs-lookup"><span data-stu-id="19b57-114">The data type of a navigation property is determined by the [multiplicity](../../../../docs/framework/data/adonet/association-end-multiplicity.md) of its remote [association end](../../../../docs/framework/data/adonet/association-end.md).</span></span> <span data-ttu-id="19b57-115">Angenommen, eine Navigationseigenschaft, `OrdersNavProp`, ist für einen `Customer`-Entitätstyp vorhanden und navigiert eine 1:n-Zuordnung zwischen `Customer` und `Order`.</span><span class="sxs-lookup"><span data-stu-id="19b57-115">For example, suppose a navigation property, `OrdersNavProp`, exists on a `Customer` entity type and navigates a one-to-many association between `Customer` and `Order`.</span></span> <span data-ttu-id="19b57-116">Da das Remotezuordnungsende für die Navigationseigenschaft eine Multiplizität von n (\*) aufweist, ist sein Datentyp eine Auflistung (von `Order`).</span><span class="sxs-lookup"><span data-stu-id="19b57-116">Because the remote association end for the navigation property has multiplicity of many (\*), its data type is a collection (of `Order`).</span></span> <span data-ttu-id="19b57-117">Ist eine Navigationseigenschaft, `CustomerNavProp`, für den `Order`-Entitätstyp vorhanden, wäre sein Datentyp dementsprechend `Customer`, da die Multiplizität des Remoteendes "eins" (1) beträgt.</span><span class="sxs-lookup"><span data-stu-id="19b57-117">Similarly, if a navigation property, `CustomerNavProp`, exists on the `Order` entity type, its data type would be `Customer`, because the multiplicity of the remote end is one (1).</span></span>
