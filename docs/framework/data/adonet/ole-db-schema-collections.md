# <a name="ole-db-schema-collections"></a>OLE DB-Schemaauflistungen

 Der Microsoft SQL Server OLE DB-Treiber unterstützt neben den allgemeinen schemaauflistungen die folgenden spezifischen schemaauflistungen:
