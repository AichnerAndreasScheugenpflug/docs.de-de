# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4-Schnittstelle
Erweitert logisch die ICorDebugAppDomain-Schnittstelle, um ein verwaltetes Objekt über einen COM callable Wrapper zu erhalten.  
  
## <a name="method"></a>Methode  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[GetObjectForCCW-Methode](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|Ruft ein verwaltetes Objekt über einen Zeiger des COM Callable Wrapper (CCW) ab.|
