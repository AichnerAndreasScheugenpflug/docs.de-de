# <a name="isymunmanagedwriter5openmaptokenstosourcespans-method"></a>ISymUnmanagedWriter5::OpenMapTokensToSourceSpans-Methode
Öffnen Sie einen spezielle benutzerdefinierte Daten im Abschnitt zum Ausgeben der Span-Token-to-Source-Zuordnungsinformationen in. Öffnen in diesem Abschnitt aus, wenn eine Methode bereits geöffnet ist, oder umgekehrt, ein Fehler wird.  
  
## <a name="syntax"></a>Syntax  
  
```idl  
HRESULT OpenMapTokensToSourceSpans();  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Gibt `HRESULT`zurück.  
  
## <a name="requirements"></a>Anforderungen  
 **Header:** CorSym.idl, CorSym.h
