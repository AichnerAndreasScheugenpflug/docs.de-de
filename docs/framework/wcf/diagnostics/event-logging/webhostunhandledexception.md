# <a name="webhostunhandledexception"></a>WebHostUnhandledException
ID: 109  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass eine nicht behandelte WebHost-Ausnahme aufgetreten ist. Das Ereignis listet die Absenderinformationen, die Ausnahme, den Prozessnamen und die Prozess-ID auf.
