# <a name="webhosthttperror"></a>WebHostHttpError
ID: 110  
  
 Schweregrad: Fehler  
  
 Kategorie: ServiceModel  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass eine WebHost-HTTP-Ausnahme aufgetreten ist. Das Ereignis listet die Absenderinformationen, die HTML-Fehlermeldung, die Ausnahme, den Prozessnamen und die Prozess-ID auf.
