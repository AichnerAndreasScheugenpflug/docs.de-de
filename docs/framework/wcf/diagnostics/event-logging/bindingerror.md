# <a name="bindingerror"></a>BindingError
ID: 159  
  
 Schweregrad: Fehler  
  
 Kategorie: SMSvcHost  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis zeigt an, dass ein Fehler beim Analysieren der angegebenen Bindung der bestimmten Website aufgetreten ist. Daher wird das Protokoll für die Website temporär deaktiviert. Das Ereignis listet die Elemente Bindung, Quelle, Ausnahme, Prozessname und Prozess-ID auf.
