# <a name="endpoint-calls-duration"></a>Endpunkt: Dauer von Aufrufen
Die durchschnittliche Dauer der Aufrufe dieses Endpunkts.  Die durchschnittliche Dauer wird anhand dieser Formel berechnet: (N1-N0)/(D1-D0).
