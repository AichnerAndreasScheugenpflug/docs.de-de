# <a name="systemservicemodeltxsourcetxscoperequiredistransactionflow"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactionFlow
Die angegebene Transaktion ist in den angegebenen Vorgang übergegangen.  
  
## <a name="description"></a>Beschreibung  
 Wird nachverfolgt, wenn die für Benutzercode verfügbar gemachte Transaktion aus einer externen Quelle kam.
