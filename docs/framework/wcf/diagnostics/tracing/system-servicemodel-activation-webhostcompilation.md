# <a name="systemservicemodelactivationwebhostcompilation"></a>System.ServiceModel.Activation.WebHostCompilation
System.ServiceModel.Activation.WebHostCompilation  
  
## <a name="description"></a>Beschreibung  
 Tritt auf, wenn ein von Internet Information Service (IIS) oder Windows Process Activation Service (WAS) gehosteter Dienst aus Inlinecode kompiliert wird, der sich in der .svc-Datei befindet.
