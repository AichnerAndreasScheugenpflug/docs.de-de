## <a name="monolithic-application-deployed-as-a-container"></a><span data-ttu-id="4dd46-137">Monolithische Anwendung, die als Container bereitgestellt wird</span><span class="sxs-lookup"><span data-stu-id="4dd46-137">Monolithic application deployed as a container</span></span>
